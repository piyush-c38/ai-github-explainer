# Recommended Libraries and Reasons

## Frontend

*   **Next.js:** (Framework) Provides a great developer experience with features like server-side rendering, static site generation, and API routes. It's a solid foundation for a modern web application.
*   **TypeScript:** (Language) Adds static typing to JavaScript, which helps in building robust and maintainable applications.
*   **Tailwind CSS:** (Styling) A utility-first CSS framework that allows for rapid UI development without writing custom CSS.
*   **React Flow:** (Visualization) A powerful library for building interactive node-based graphs. It's perfect for our visualization features.
*   **Mermaid:** (Diagramming) A simple and effective way to generate diagrams from text. It's ideal for the architecture overview diagram.
*   **SWR / React Query:** (Data Fetching) These libraries provide a great way to manage data fetching, caching, and revalidation, which will be crucial for our dashboard.
*   **Zustand:** (State Management) A small, fast, and scalable state management solution for React. It's a good choice if we need something more powerful than React's built-in state management.

## Backend

*   **Express.js:** (Framework) A minimal and flexible Node.js web application framework. It's a great choice for building our REST API.
*   **TypeScript:** (Language) For the same reasons as the frontend - robustness and maintainability.
*   **Groq:** (LLM API) Provides fast inference for large language models, which is essential for a responsive chat experience.
*   **ChromaDB:** (Vector DB) An open-source vector database that is easy to set up and use locally. It's a good starting point for our RAG pipeline.
*   **`simple-git`:** (Git Client) A lightweight and easy-to-use library for interacting with Git repositories from Node.js.
*   **`tree-sitter`:** (Parser) A powerful and efficient parser generator that supports a wide range of programming languages.
*   **Babel / TypeScript Compiler API:** (JS/TS Analysis) For deep and accurate analysis of JavaScript and TypeScript codebases.

## Monorepo

*   **Turborepo:** (Build System) A high-performance build system for JavaScript and TypeScript monorepos. It will help us manage our monorepo efficiently.
